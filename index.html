1. What Does javascript event loop do?

1. javascript event loop works like for/while loop & also priority base like which event come first.
2. Event loop firstly take data or event from stact then run it.
3. If event loop cannot find any data or event from stact then it takes data from queue.
4. The event loop is a constantly running process that monitors both the callback queue and the call stack.
5. If the call stack is not empty, the event loop waits until it is empty, and places the next function from the callback queue to the call stack.
If the callback queue is empty, nothing will happen.

2. Difference between Local Storage & Session Storage?

Local Storage :

1. For local storage data is deleted by its own users
2. Local storage works for all tabs
3. Local Storage's Storage file size usually 5/10 MB

Session Storage :

1. For Session storage data dont need to be deleted by user beacuse when browser is close data will be automatically deleted
2. A page session lasts as long as the tab or the browser is open, and survives over page reloads and restores.
3. Session Storage's Storage size usually 5MB
